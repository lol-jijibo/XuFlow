# Apple-Design Chat Input Redesign

## Context

Redesign the chat input area in ChatPanel.vue with Apple-inspired aesthetics: frosted glass container, clean layout, floating extension cards instead of cluttered toolbars. Single-file change.

## Changes

### Script (minor addition)
- Add `activeExt` ref (`'model' | 'memory' | 'tools' | null`) to control floating extension card visibility

### Template — Replace `.chat-footer` section
```
┌───────────────────────────────────────┐
│  📄 Xuflow · 本地          (badge)   │  ← context-badge at top
│                                       │
│  向 Agent 下达指令...  (placeholder)   │  ← NInput, transparent bg
│                                       │
│ [🤖模型] [🧠记忆] [🔗工具]   [↑]     │  ← ext pills (left) + send arrow (right)
│                                       │
│ ┌─ Floating card (v-if activeExt) ──┐ │
│ │  Option items                      │ │
│ └────────────────────────────────────┘ │
└───────────────────────────────────────┘
```

### CSS — Full rewrite of footer styles
**Container**:
- `background: rgba(255,255,255,0.72)` + `backdrop-filter: blur(24px)` (frosted glass)
- `border: 1px solid rgba(0,0,0,0.05)` → subtle edge glow
- `border-radius: 24px` (rounded-3xl)
- `box-shadow` with `rgba(0,0,0,0.04)` → floating feel
- Dark: `rgba(28,28,33,0.84)` + `border: rgba(255,255,255,0.08)`

**Context badge**:
- Positioned top-left inside padding area
- `background: rgba(0,0,0,0.04)` capsule, `border-radius: 20px`
- SF Pro font, 12px, muted color
- Folder icon + "Xuflow · 本地" text

**Input area**:
- NInput with all borders/backgrounds hidden via `:deep()` selectors
- `font-size: 16px`, `line-height: 1.6`, Apple system font stack
- Placeholder: `color: #aaa`, "向 Agent 下达指令..."

**Extension pills** (bottom-left):
- `[🤖 模型] [🧠 记忆] [🔗 工具]` as rounded pills
- `bg-black/5`, `rounded-full`, small text
- Active state: slight background shift + floating card appears

**Send button** (bottom-right):
- Circular, `bg-black/10` → hover to `bg-blue-500`
- Disabled: `opacity: 0.25`
- Up arrow SVG icon, `stroke-width: 2`

**Floating extension card**:
- Appears below the input container
- `background: rgba(255,255,255,0.9)` + `backdrop-filter: blur(20px)`
- Rounded, small shadow, positioned absolutely or in-flow
- Menu items styled cleanly

## Verification

1. Run `cd desktop && npm run dev` (or `pnpm tauri dev`)
2. Check light mode: frosted glass effect, clean layout
3. Check dark mode: dark frosted glass
4. Type text → send button becomes active (full opacity)
5. Click extension pills → floating card appears/disappears
6. Press Enter → message sends, input clears
