# 聊天框底部输入区域重构计划

## Context
用户要求重构 ChatPanel 底部输入区域，去掉当前的三层嵌套结构 (`chat-footer > input-container > input-wrapper`)，改为扁平的单层底部栏。底部栏分为上下两行：上行为文本输入区，下行为工具栏（+号添加文件、权限模式切换、模型名、发送按钮）。

## 修改文件
**`desktop/src/components/chat/ChatPanel.vue`** — 唯一需要修改的文件

## 具体改动

### 1. 模板重构
移除当前的三层嵌套，替换为:

```
chat-footer (单层容器)
  ├─ input-area (上行)
  │   └─ textarea / NInput (用户输入文本 + 未来放置图片/附件缩略图)
  └─ input-toolbar (下行)
      ├─ .toolbar-left
      │   ├─ + 按钮 (添加图片/文件，目前只用 icon 占位)
      │   └─ 权限模式切换按钮 (切换: auto / approval 两种模式)
      └─ .toolbar-right
          ├─ 模型名 (只显示文字，来自 configStore.activeModelName)
          └─ 发送按钮 (圆形，仅向上箭头 icon)
```

### 2. 新增响应式状态
```ts
// 权限模式: 'auto' = 完全自动执行, 'approval' = 请求用户确认
const permissionMode = ref<'auto' | 'approval'>('approval')
```

### 3. CSS 样式
- `.chat-footer` — 统一背景色，与 `.chat-panel` 一致，`padding: 16px 24px 20px`，无边框无分隔线
- `.input-area` — 上方的输入区域，min-height 撑开，无边框，透明背景
- `.input-toolbar` — 下方工具栏，`display: flex; justify-content: space-between; align-items: center`
- 权限按钮 — 小型切换按钮，auto 模式用绿色/gray 状态标识，approval 模式用黄色/警告色
- 模型名 — 小号 muted 文字
- 发送按钮 — 圆形 `32x32`，主色调，内含向上箭头 SVG

### 4. 保留功能
- Enter 发送、Shift+Enter 换行
- 发送中 loading 状态（发送按钮 loading 或变色）
- 运行中停止按钮（工具栏中显示停止按钮替代发送按钮）
- 空消息时 welcome 页面不变

## 验证方式
1. 启动应用 `npm run tauri dev`，观察聊天框底部区域
2. 确认输入区域与聊天面板背景统一，无分层感
3. 确认工具栏布局正确：+号 | 权限切换 | 模型名 | 发送箭头
4. 确认深色/浅色模式均正常
5. 确认 Enter 发送、Shift+Enter 换行正常工作
