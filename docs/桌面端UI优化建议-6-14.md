## 二、用户体验优化建议

1. **窗口状态记忆** — 记住窗口大小/位置，下次启动恢复
2. **深色/浅色主题切换** — NaiveUI 原生支持，只需加一个主题 store
3. **消息搜索** — 在会话中搜索历史消息内容
4. **会话导出** — 导出为 Markdown/PDF
5. **快捷键系统** — `Ctrl+N` 新会话、`Ctrl+Enter` 发送、`Ctrl+K` 命令面板
6. **输入框增强** — 多行输入、`@` 引用文件、粘贴图片
7. **消息操作** — 复制单条消息、重新生成、编辑后重发
8. **空状态引导** — 新建项目/会话后的引导页面
9. **文件拖拽** — 拖拽文件到聊天窗口作为上下文
10. **流式输出打字效果优化** — 当前流式渲染可能有闪烁，可加 debounce

## 三、可新增功能

- **项目文件树浏览** — 侧边栏展示项目目录结构
- **终端集成** — 内嵌终端面板直接执行命令
- **代码差异对比** — 工具修改文件后展示 diff
- **Agent 工具使用统计** — 展示 token 消耗、工具调用次数
- **多 Agent 会话** — 并行运行多个 Agent 对话
- **本地模型支持** — 接入 Ollama 等本地模型
- **插件系统** — 自定义工具扩展
